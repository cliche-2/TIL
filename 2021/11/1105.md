## 🗒️압축/해제 OP

```bash
# 압축
> FORMAT -cvf FILENAME TARGET_DIR
> jar -cvf aaa.jar .

# 해제
> FORMAT -xvf TARGET_FILE
> jar -xvf aaa.jar
```
